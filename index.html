⬆⬇ Sincronización de datos

Los datos se guardan localmente en el navegador. Usa estas opciones para hacer copia de seguridad o transferir datos entre dispositivos mediante un archivo JSON.

📥 EXPORTAR datos

Descarga un archivo JSON con todas las incidencias. Súbelo manualmente a tu repositorio GitHub.

📤 IMPORTAR datos

Carga un archivo JSON previamente exportado. Sustituirá los datos actuales.

💡 Flujo recomendado con GitHub:
1. Desde el móvil: registra incidencias → exporta JSON
2. Sube el JSON a tu repositorio GitHub (reemplaza el archivo)
3. Desde el PC: descarga el JSON del repo → importa en REGIN
4. Trabaja el parte → marca incidencias como pasadas → exporta
5. Sube el JSON actualizado a GitHub
